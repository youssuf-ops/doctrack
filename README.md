# DocTrack — Document Management Platform

A modern document ownership and management dashboard built with React 19, TypeScript strict mode, and TanStack ecosystem.

## Live Demo
[doctrack-eta.vercel.app](https://doctrack-eta.vercel.app)

## Tech Stack

- **React 19** — UI library
- **TypeScript** (strict mode) — type safety across the entire codebase
- **Vite 8** — build tool and dev server
- **TanStack Query** — async data fetching with intelligent caching
- **Zod** — runtime schema validation
- **Ant Design** — enterprise UI components
- **Tailwind CSS v4** — utility-first styling

## Features

- Dashboard with real-time document metrics
- Document detail page with complete ownership trail (audit log)
- Upload form with Zod validation and real-time field feedback
- Dark theme throughout
- Responsive layout with collapsible sidebar
